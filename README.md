# Linux Programming

Linux 시스템 프로그래밍을 학습하며 작성한 C 프로젝트 모음입니다.  
프로세스 관리, 파일 입출력, 리다이렉션 등 Unix/Linux 환경의 핵심 개념을 직접 구현하는 것을 목표로 합니다.

This repository contains Linux system programming projects written in C.  
It focuses on core Unix/Linux concepts such as process control, file I/O, redirection, and basic shell behavior.

## Environment
- Language: C
- OS: Linux (Ubuntu)
- Compiler: gcc
